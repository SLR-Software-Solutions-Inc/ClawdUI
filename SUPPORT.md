# Support

ClawdUI is an independent, open-source project maintained by SLR Software Solutions Inc.

## Report a bug or request a feature

Please open an issue on GitHub:

> https://github.com/SLR-Software-Solutions-Inc/ClawdUI/issues

Before filing:

- Search existing issues — yours may already be tracked.
- Include your OS + version, ClawdUI version (Help → About), and `claude --version`.
- For crashes, attach the sidecar log (`~/Library/Logs/ClawdUI/sidecar.log` on macOS; `%LOCALAPPDATA%\ClawdUI\logs\` on Windows; `~/.local/state/clawdui/logs/` on Linux).

## Security issues

For anything that could compromise users (credential disclosure, RCE, sandbox escape) please **do not** open a public issue. Use GitHub's private security advisory flow on the repo:

> https://github.com/SLR-Software-Solutions-Inc/ClawdUI/security/advisories/new

## Discussions & general questions

Use [GitHub Discussions](https://github.com/SLR-Software-Solutions-Inc/ClawdUI/discussions) for usage questions, configuration help, and feature brainstorming. There is no Slack, Discord, or private support email — keeping conversation public means everyone benefits from the answer.

## Contributing

Pull requests are welcome. For non-trivial changes please open an issue or discussion first so we can agree on the shape of the change before you spend time on it. See `CONTRIBUTING.md` (if present) for the contributor checklist.

## Support development

ClawdUI is funded by its maintainers' time. If it saves you time, helps your workflow, or you just want to keep the lights on, a one-off contribution goes a long way toward hosting, signing certificates, and time spent on new features. Donations are entirely voluntary and never gate any feature.
